What is Multi-Threading?

Multi Threading is a technique by which a single set of code can be used by several processors at different stages of execution. We ussually use it when we want to execute 2 or more processes at the same time. Initially when a program runs it has only one thread. We can add more threads to our program by using multi-threading.

How to do multi-threading in python using threading module

What if I want to print 2 things at the same time?

for i in range (10): print("Hello")

for i in range (10): print("Hi")

How to do it using Multi-Threading

We will create two function we want to execute and then we will assign those function to threads.

Creating functions

def hello():
for i in range (10): print("Hello")

def hi():
for i in range (10): print("Hi")

Assigning the functions to threads

For this we will require the threading module

import threading


hello_thread = threading.Thread(target=hello)
hi_thread = threading.Thread(target=hi)
hello_thread.start()
hi_thread.start()

Output :

Hello Hi Hello Hi Hello Hi Hello Hi Hello Hi Hello Hi Hello Hi Hello Hi Hello Hi Hello Hi Hello Hi

The output shows that both the works were getting executed at the same time. As we started the hello_thread first it printed hello first and hi after hello.
