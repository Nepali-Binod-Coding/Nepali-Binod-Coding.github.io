Programming languages to learn in 2020

Every programming language has its own features and roles which they are known and used for. some easy examples is that C++ is mainly used for game development as most of the free and amazing game engines use C++ as their main language some of theese game engines are Unity, Unreal Engine, Godot etc..

Game-Development

C++ or C#

These languages are used for development of high-end games. These games require a really powerful game engine to be created. These games are cross-platform games because of these game engines which make the code useable on every operating system.

Data-Science or AI

Python or R

These languages are used for AI development and Data-Science because of presence of some strong libraries and frameworks. They are easy to learn also. These languages are slow but the programmer is ussually faster in the case of these languages.


Android development

Kotlin or Flutter

The best language for android right now is Kotlin as it is officially the native language of Android. Earlier Java was the native language off Android. Flutter is also recomended but it is not the native language but it can run on both IOS and Android as it is a cross-platform language.

Web development

HTML5, CSS3, JAVASCRIPT

These are the most basic, popular, easy languages. These languages are the must learn languages even if you donot belong to the web development feild. HTML is used as the skeleton for the website, CSS is the decoration of the website and JAVASCRIPT is used as the brain.
